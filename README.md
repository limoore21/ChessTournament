# Chess Tournament Manager

Система для управления шахматными турнирами на локальном хосте. 

Текущая стадия: реализована архитектура MVC и CRUD управление игроками.

## Что уже реализовано
[1] Проектирование базы данных (5 таблиц).

[2] Подключение к БД через PDO.

[3] Базовый контроллер (`index.php`) с роутингом действий.

[4] Модель игроков (CRUD: добавление, просмотр, удаление).

[5] Главный шаблон верстки (`layout.php`).

## Как запустить
1. Склонируйте репозиторий.
2. Импортируйте базу данных из файла database(пока не приложен и работает только на MariaDB, ниже приложил sql код адаптированный под mysql)
3. Настройте параметры подключения в `config.php`.
4. Запустите встроенный сервер PHP.

## В планах
- Реализация алгоритма жеребьевки.
- Генерация турнирной сетки (1/4, 1/2, Финал).
- Логика продвижения победителей по сетке.

## 3. Техническое задание (ТЗ)
Приложение для автоматизации проведения шахматных турниров по системе Single Elimination (на вылет).


1.  **Управление участниками:**
    * Возможность регистрации неограниченного количества игроков в общей базе
    * Редактирование данных игрока
    * Удаление игроков (с каскадным удалением их истории игр)
2.  **Турнирный менеджер:**
    * Жеребьевка участников (рандомное распределение по парам).
    * Поддержка турниров на 6-8 человек.
    * Интерактивное управление результатами (выбор победителя в матче).
    * Автоматическое продвижение победителя в следующий раунд.
3.  **Интерфейс:**
    * Чистый HTML/CSS дизайн без лишних перегруженных библиотек.
    * Визуальное отображение сетки в виде колонок-раундов.
